# lodash initial

::: tip
获取数组 `array` 中除了最后一个元素之外的所有元素。
:::

## 使用

```javascript
_.initial(array)
```

获取数组 `array` 中除了最后一个元素之外的所有元素（注：去除数组 array 中的最后一个元素）。

## 引入版本

0.1.0

## 参数

1. `array` (Array): 要查询的数组。

## 返回

(Array): 返回截取后的数组 `array`。

## 例子

```javascript
_.initial([1, 2, 3]);
// => [1, 2]
```
